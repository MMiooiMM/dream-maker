# ABO 世界觀說明文件（Omegaverse）

## 目的與使用方式
本文件提供一套可直接套用於創作的 ABO（Omegaverse）世界觀規則。重點在於「可組合、可調參數化」，方便作者依故事風格與尺度調整，並與 DreamMaker 的設定欄位對應。

## 核心前提與基礎名詞
- 第一性別：常見為男性/女性（或你的世界觀自訂性別），影響生理與社會角色，但不是 ABO 的核心。
- 第二性別：Alpha / Beta / Omega，決定信息素相性、發情期反應、社會階層與婚配傾向。
- 信息素：可被嗅覺或身體感知的生理訊號，影響情緒、吸引力、社會互動與權力關係。

## 第二性別總覽
- Alpha：通常被視為領導型或攻擊性較強，但也可能被期待承擔保護與責任。
- Beta：被視為中性/穩定群體，通常不觸發信息素極端反應，也常被定位為社會秩序的「普通人」。
- Omega：被視為高敏感體質，與信息素反應最強，社會常賦予「需要被保護」的刻板印象。

## 生理機制總覽
- 發情期/易感期：Omega（或依世界觀設定其他性別）在周期性階段更易受信息素影響，情緒與身體反應增強。
- 求偶期：Alpha 在特定刺激下出現更強的求偶與占有傾向，表現為信息素釋放增強或行為明顯。
- 築巢：部分 Omega 可能在高壓或高情緒時期表現出築巢行為，象徵安全感需求。
- 結合與標記：標記通常代表強烈關係綁定，可是自願、半自願或被迫；視世界觀可決定標記是否具有生理束縛或法律效力。

## 生育與繁衍規則
- 男孕可選：可設定 Alpha/ Omega 皆具繁衍能力，或僅特定性別能孕育。
- 繁衍條件：可設定是否需要標記、是否僅在發情期可受孕、或是否可透過醫療介入。
- 社會影響：高生育率可能帶來 Omega 的社會壓力或保護政策，也可導致階級化。

## Alpha 強弱分級規則
### 分級軸線
- 信息素壓制力：對他人的壓迫感與影響範圍。
- 社會支配力：權勢、資源與話語權的綜合體現。
- 自我控制力：能否克制本能與情緒反應。

### 強A
- 描述：信息素壓制力強、社會支配力高，且通常自我控制力高或被迫外顯控制。
- 寫作用途：適合權力型衝突、追妻火葬場、上位者自我破防或保護型反差。

### 中A
- 描述：信息素與社會資源中等，能影響局勢但不具壓倒性優勢，自我控制力視角色性格波動。
- 寫作用途：適合可塑性高的角色弧線，如成長型 Alpha 或在壓力中失控再自省。

### 弱A
- 描述：信息素壓制力低或社會資源受限，自我控制力通常較強以避免被邊緣化。
- 寫作用途：適合逆襲、反差、或被刻板印象壓制的情節；可創造「不靠本能靠意志」的魅力。

## Omega 分級規則
- 高敏感 Omega：信息素反應極強，容易引發周遭 Alpha 反應；適合高張力、易失控的情感戲。
- 中等 Omega：生理反應可控，能在理性與本能間拉扯；適合平衡情感與劇情推進。
- 低敏感 Omega：反應偏弱，常被誤判為 Beta 或難以觸發標記；適合身份錯位或延遲揭露劇情。

## Beta 定位與變體規則
- Beta 通常不受強烈信息素牽引，適合扮演社會秩序維持者、醫療人員或中立派。
- 變體 Beta 可設定對信息素部分敏感，或在特定對象下出現例外反應，作為特殊設定或反轉。

## 抑制與醫療
- 抑制劑：降低發情/求偶反應，常用於社會秩序維持或自我控制。
- 阻斷劑：降低信息素感知，降低彼此吸引或壓制效果。
- 偽裝/掩味：以香氣、藥劑或裝置遮蔽信息素，常用於身分隱藏或社會歧視情節。

## 社會制度與階層
- 法律：可設定標記是否具有婚姻或監護效力，或需要明確同意才能成立。
- 職場：強 Alpha 可能天然擁有上升優勢，Omega 可能面臨偏見或保護性歧視。
- 婚配制度：可設定包辦、配對機制、或自由婚戀文化。
- 歧視與特權：Omega 的保護政策可能帶來依附性壓力；Alpha 的特權可能導致反感或政治動盪。

## 關係動力與衝突來源
- 強烈吸引：信息素相性造成理智被挑戰。
- 抗拒與防衛：角色明知吸引卻抵抗，形成情感張力。
- 標記爭議：被迫標記、契約標記、或「標記不代表愛」的價值衝突。
- 社會壓力：家庭安排、階級制或媒體輿論成為外部阻力。

## 可調世界觀參數清單
- 信息素可感知範圍（僅近距離或可遠距影響）。
- 標記是否可解除、是否有法律效力。
- 生育是否必須標記、是否能醫療介入。
- Alpha 強弱分級是否影響法律或社會地位。
- Omega 是否具特殊職業限制或保護制度。

## 創作建議與倫理邊界
- 同意原則：即便設定有本能壓力，也建議呈現角色的選擇與後果。
- 權力不對等：若角色社會地位差距大，需呈現其衝突與心理掙扎，避免合理化壓迫。
- 反刻板印象：可塑造弱A的堅韌或高位Omega的主動性，增強角色層次。

## 與 DreamMaker 設定對應
- `aboEnabled`：啟用/停用 ABO 世界觀設定。
- `aboSecondGender`：角色第二性別（Alpha / Beta / Omega）。
- `pheromone`：可作為世界觀阻力來源之一（信息素引發衝突）。

## 附錄：非主流變體（可選）
- Enigma：極端不穩定信息素體質，可能造成大範圍影響。
- Delta：介於 Alpha/Beta 的變體，對信息素反應較弱但具權力傾向。
- Zeta：被設定為幾乎不受信息素影響的特殊族群。
